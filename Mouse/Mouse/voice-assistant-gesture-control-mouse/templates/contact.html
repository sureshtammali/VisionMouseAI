Frequently Asked Questions

How do I set up the gesture control system?

Make sure you have a working webcam and required Python packages installed. Click "Start Gesture" on the home page to begin gesture recognition.

What voice commands are supported?

The system supports 40+ commands starting with "pixel" including time, date, search, media control, and system functions. Check the home page for a complete list.

Can I customize the gestures?

Currently, the system uses predefined gestures. Custom gesture training is planned for future updates. You can modify the gesture recognition code for custom implementations.

What are the system requirements?

You need Python 3.7+, a webcam, microphone, and required packages (OpenCV, MediaPipe, Flask, etc.). The system works on Windows, Linux, and macOS.
