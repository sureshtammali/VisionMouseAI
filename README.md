# 📌 Project Overview

**VisionMouseAI** is an advanced computer vision-based virtual mouse system that enables users to control the computer cursor using hand gestures instead of a physical mouse.

The system uses real-time video capture and hand tracking techniques to detect finger movements and translate them into cursor actions such as movement, clicking, dragging, and scrolling.

This project demonstrates the application of **Artificial Intelligence and Computer Vision** in developing touchless human-computer interaction systems.
